# 內湖淹水風險判讀儀表板

這是一個單頁式決策儀表板，用於整理 2026-06-26 強降雨事件中，內湖淹水可能成因的初步判讀。

## 主要結論

現有資料較不支持「內湖本地極端暴雨」或「基隆河主河道達警戒倒灌」作為單一主因；較合理的初步判斷是長時間降雨後，整個集水區逕流同時進入都市排水系統，造成內水排除不及或流域排水瓶頸。

## 資料線索

- 內湖本地雨強約 20-24 mm/hr，未達極端短延時暴雨。
- 文山、信義、南港山區長時間降雨，形成上游產流壓力。
- 南湖大橋、大直橋等相關河川水位未達三級警戒。
- 台北市抽水站公開 API 僅提供最新快照，不能取代歷史逐分鐘運轉紀錄。
